# 欧几里得算法

又称辗转相除法，迭代求两数 gcd。

由 (a,b)=(a,ka+b)(a, b) = (a, ka + b)(a,b)=(a,ka+b) 的性质，gcd⁡(a,b)=gcd⁡(b,amodb)\gcd(a, b) = \gcd(b, a\bmod b)gcd(a,b)=gcd(b,amodb)。容易证明这么做的复杂度是 O(log⁡n)O(\log n)O(logn)。

注意：gcd⁡(0,a)=a\gcd(0, a) = agcd(0,a)=a。

# 裴蜀定理

裴蜀定理：设 (a,b)=d(a, b) = d(a,b)=d，则对任意整数 x,yx, yx,y，有 d∣(ax+by)d|(ax + by)d∣(ax+by) 成立；特别地，一定存在 x,yx, yx,y 满足 ax+by=dax + by = dax+by=d。

等价的表述：不定方程 ax+by=c(a,b,c为整数)ax + by = c(a, b, c 为整数)ax+by=c(a,b,c为整数) 有解的充要条件为 (a,b)∣c(a, b)|c(a,b)∣c。

# 扩展欧几里得（exgcd）

裴蜀定理的推论：a,ba, ba,b 互质等价于 ax+by=1ax + by = 1ax+by=1 有解。

因此，要求不定方程 ax+by=cax+by=cax+by=c 的解，只需求 ax+by=d,d=gcd⁡(a,b)ax+by=d,d=\gcd(a,b)ax+by=d,d=gcd(a,b) 的解，然后扩倍（乘上 cd\frac{c}{d}dc​）即可。
