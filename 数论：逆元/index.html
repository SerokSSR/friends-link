# 逆元

# 什么是逆元？

若 ax≡1(modb)ax \equiv 1 \pmod bax≡1(modb)，则称 xxx 是 aaa 关于模 bbb 的逆元，常记做 a−1a^{−1}a−1。

上式等价于 ax+by=1ax + by = 1ax+by=1，因此，一种求逆元的方法就是利用扩欧解方程 ax+by=1ax + by = 1ax+by=1。

显然，逆元不一定存在：其存在的充要条件为 (a,b)=1(a, b) = 1(a,b)=1。

推论：ppp 是质数，ppp 不整除 aaa，则 aaa 模 ppp 的逆元存在。

结论：在 [0,b)[0, b)[0,b) 的范围内，aaa 关于模 bbb 的逆元（若存在）是唯一的。

证明：
反证法，若 aaa 有两个逆元 0<x1<x2<b0 < x_1 < x_2 < b0<x1​<x2​<b，
即 ax1≡ax2≡1(modb)ax_1 ≡ ax_2 ≡ 1 \pmod bax1​≡ax2​≡1(modb)，
那么有 b∣a(x2−x1)b|a(x_2 − x_1)b∣a(x2​−x1​) 成立。
又由于 (a,b)=1(a, b) = 1(a,b)=1，因此 b∣(x2−x1)b|(x_2 − x_1)b∣(x2​−x1​)。
其中 0<x2−x1<b0 < x_2 − x_1 < b0<x2​−x1​<b，产生了矛盾。

# 求解逆元

ax≡1(modp)ax\equiv 1 \pmod pax≡1(modp)

# 扩欧

上面已经提到过了，直接解方程 ax+py=1ax + py = 1ax+py=1 即可。这个方法十分容易理解，而且对于单个查找效率不错，尤其是在模数比较大的时候。

这个做法在 a,pa,pa,p 互质，但 ppp 不是质数时也可以使用。其他方法均要求 ppp 必须为素数。

# 快速幂

这个做法要利用费马小定理：若 ppp 为素数，aaa 为正整数，且 a,pa,pa,p 互质，则有

ap−1≡1(modp)a^{p-1} \equiv 1 \pmod {p}ap−1≡1(modp)

因此，

ax≡1(modp)⇒ax≡ap−1(modp)⇒x≡ap−2(modp)\begin{aligned} ax&\equiv 1 &\pmod p\\ \Rightarrow ax&\equiv a^{p-1} &\pmod p\\ \Rightarrow x &\equiv a^{p-2} &\pmod p \end{aligned}ax⇒ax⇒x​≡1≡ap−1≡ap−2​(modp)(modp)(modp)​

用快速幂算出 ap−2(modp)a^{p-2} \pmod pap−2(modp) 的值，这个数就是它的逆元了。

# 递推

用于求连续的数对于同一模数的逆元。

# P3811 【模板】乘法逆元

首先有 1−1≡1(modp)1^{-1}\equiv 1 \pmod p1−1≡1(modp)。

然后设 p=ki+r(1<r<i<p)p=ki+r(1<r<i<p)p=ki+r(1<r<i<p)，即 kkk 是 pi\frac{p}{i}ip​ 的商，rrr 是余数。

再将这个式子放到模 ppp 意义下，有

k×i+r≡0(modp)k\times i+r \equiv 0 \pmod pk×i+r≡0(modp)

两边乘上 i−1r−1i^{-1}r^{-1}i−1r−1，有

k×r−1+i−1≡0(modp)⇒i−1≡−k×r−1(modp)⇒i−1≡−⌊pi⌋∗(pmodi)−1(modp)\begin{aligned} k\times r^{-1}+i^{-1}&\equiv 0 &\pmod p\\ \Rightarrow{}i^{-1}&\equiv -k\times r^{-1} &\pmod p\\ \Rightarrow{}i^{-1}&\equiv -\lfloor \frac{p}{i} \rfloor*(p \bmod i)^{-1} &\pmod p \end{aligned}k×r−1+i−1⇒i−1⇒i−1​≡0≡−k×r−1≡−⌊ip​⌋∗(pmodi)−1​(modp)(modp)(modp)​

于是，我们就可以递推求得逆元了。

# 阶乘

首先有这样一个结论：1n!=1n×1(n−1)!\frac{1}{n!}=\frac{1}{n}\times\frac{1}{(n-1)!}n!1​=n1​×(n−1)!1​，即 (n−1)!(n-1)!(n−1)! 的逆元等于 n!n!n! 的逆元乘 nnn。
